# ENSO Prediction using Deep Learning

Replicación parcial del modelo CNN de Ham et al. (2019) para predicción 
multi-año de ENSO usando datos CMIP5 — IMARPE/PUCP

## Estructura
- `data/` — Muestras del índice Nino3.4 (train/validation)
- `scripts/` — Scripts de reproducción de figuras (Figure2a, 2b, 3a)
- `figures/` — Gráficos generados

## Referencia
Ham et al. (2019). Deep learning for multi-year ENSO forecasts. *Nature*, 573.
https://doi.org/10.1038/s41586-019-1559-7
